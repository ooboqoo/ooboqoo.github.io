# macOS 使用笔记

macOS 是基于 FreeBSD 的操作系统。

## 命令行

```bash
$ sudo shutdown -h now  # 立即关机
$ sudo shutdown -h +10  # 10分钟后关机
$ sudo halt             # sleep 吗？
$ sudo reboot           # 重启
$ sudo shutdown -r now  # 重启

$ sudo -s  # 临时升级到系统管理员
```
