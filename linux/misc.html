开启 xterm 终端 256色和终端下 vim 256色

相同的colorschema,vim和gvim的颜色差距还是很大的，因为gvim使用X的颜色，而vim只能使用终端提供的颜色，所以造成了二者的显示差异。

xterm开启256色

现在的终端模拟器早就支持256色了，不过默认可能还是8色的。

~/.bashrc文件添加
if [ "$TERM" == "xterm" ]; then
export TERM=xterm-256color
fi

$ echo $TERM  // 验证是否设置成功
xterm-256color

vim开启256色支持

编辑 ~/.vimrc 文件，添加set t_Co=256 t_Co 即 terminal Color 之意。
