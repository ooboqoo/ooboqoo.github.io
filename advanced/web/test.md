Protractor 是一个 end-to-end 的测试框架，它的构建基于 Selenium WebDriver 之上。

Selenium 是一个 Web 的自动化测试工具，最初是为网站自动化测试而开发的，有点像我们玩游戏用的按键精灵，可以按指定的命令自动操作，不同是 Selenium 可以直接运行在浏览器上，它支持所有主流的浏览器（包括 PhantomJS 这些无界面的浏览器）。
