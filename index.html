站点结构介绍

前端基础：HTML5 + CSS3 + JavaScript 基础。 速查手册：手册虽没什么技术含量，但超级好用！ 电子书：阮一峰的《ES6 入门》 和 Nicholas 的《Understanding ECMAScript 6》，方便自己阅读。 ES & TS：ES5.1 + ES 2015 + TypeScript 笔记。 AngularJS：AngularJS + Bootstrap + jQuery + Chart.js + D3 + three.js 等。 Angular：主要是官方文档的一些简化笔记，另外有补充一些额外内容。 React：React + Redux + React Router + Flow 等。 混合应用：Electron + React Native + Ionic2 等。 软件工程：自动化、测试、及各种相关工具的笔记。 服务器：PHP + SQL + NodeJS + Koa2 + MongoDB 等。 工具箱：编辑器及 Linux 方面的笔记。

注：站点主导航条平时都隐藏在页面顶部，鼠标移到页面顶部即可调出导航条。
