站点结构介绍

前端基础：最早学 html + css + javascript 时整理的笔记，比较基础。 HTML5 手册：包含了 html5 + css3 笔记和一份前端速查手册，手册虽没什么技术含量，但超级好用！ ES6 电子书：阮一峰的《ES6 入门》 和 Nicholas 的《Understanding ECMAScript 6》，牛人出品不解释。 TypeScript：ECMAScript 2015 + TypeScript2 学习笔记。 Angular2：主要是官方文档的一些简化笔记，另外有补充一些额外内容。 Angular2 生态圈：Angular2 相关项目的资料（ Material2 + PrimeNG + Ionic2 等 ） 软件工程：自动化、测试、及各种相关工具的笔记。 服务器端：PHP + SQL + NodeJS，很久没有更新，估计格式什么的也会有问题。 工具箱：编辑器及 Linux 方面的笔记。

注：站点主导航条平时都隐藏在页面顶部，鼠标移到页面顶部即可调出导航条。

HTML5 & CSS3 小练习

Canvas：包含一个图片放大显示组件和一个弹跳小球练习，顺便做成了离线应用。 跨域访问：平时难免碰到跨域问题，就系统学习了下。 百度地图 API：学习 html5 geolocation 时发现并没什么鸟用，就玩了玩百度地图。 高温监控系统：大概试了试 websocket.io，暂时不会深入学习 Nodejs。 仿小米首页：学习 Bootstrap V4 时，顺便拿小米首页练练手，做页面更多的是拼设计和用户体验吧。

源码地址：https://github.com/ooboqoo/mdreader
