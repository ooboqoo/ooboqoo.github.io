站点结构介绍

前端教程：最早学 html + css + javascript 时整理的笔记，比较基础。 HTML5手册：包含了 html5 + css3 笔记和一份前端速查手册，手册虽没什么技术含量，但超级好用！ ES6电子书：阮一峰的《ES6 入门》 和 Nicholas 的《Understanding ECMAScript 6》，牛人出品不解释。 Angular2：ES6 + TypeScript + Angular2 相关笔记，正学习中，应该是未来的主攻方向。 软件工程：自动化、测试、及各种相关工具的笔记。 PHP & SQL：主攻前端，PHP 为辅助，这部分笔记有点历史了，这两天再整理整理。 工具箱：编辑器及 Linux 方面的笔记，玩 Linux 历史比较久了，所以玩起来还是比较轻松。

注：站点主导航条平时都隐藏在页面顶部，鼠标移到页面顶部即可调出导航条。

学习中做的几个小练习

Canvas：包含一个图片放大显示组件和一个弹跳小球练习，顺便做成了离线应用。 跨域访问：平时难免碰到跨域问题，就系统学习了下。 百度地图 API：学习 html5 geolocation 时发现并没什么鸟用，就玩了玩百度地图。 高温监控系统：大概试了试 websocket.io，暂时不会深入学习 Nodejs。 仿小米首页：学习 Bootstrap V4 时，顺便拿小米首页练练手，做页面更多的是拼设计和用户体验吧。

小项目 mdReader

源码地址：https://github.com/ooboqoo/mdreader

在本站代码的基础上做了精简和重构，做成了一个小巧的 markdown 文档阅读器。

尽量做到了精简并保持独立性。使用不需要借助 Node.js，如果要修改，也是直接 js + css。
