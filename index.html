站点结构介绍

前端基础：最早学 html + css + javascript 时整理的笔记，比较基础。 速查手册：包含了 html5 + css3 笔记和一份前端速查手册，手册虽没什么技术含量，但超级好用！ ES6 电子书：阮一峰的《ES6 入门》 和 Nicholas 的《Understanding ECMAScript 6》，牛人出品不解释。 TypeScript：ECMAScript 2015 + TypeScript2 学习笔记。 Angular2：主要是官方文档的一些简化笔记，另外有补充一些额外内容。 Angular2 周边：Angular2 相关项目的资料 ( Material2 + PrimeNG + Ionic2 等 )。 软件工程：自动化、测试、及各种相关工具的笔记。 服务器：PHP + SQL + NodeJS。 优质项目：Chart.js + D3 + three.js 等。 工具箱：编辑器及 Linux 方面的笔记。

注：站点主导航条平时都隐藏在页面顶部，鼠标移到页面顶部即可调出导航条。
