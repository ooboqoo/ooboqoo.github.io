# Linux 命令速查表


## 命令

```bash
$ rm -rf <dir>   # 删除目录
```

```bash
$ date [+format]  # 查看日期
$ cal             # 日历
```

```
$ uptime  # 开机运行时间
$ w       # display who is logged in and what they are doing

$ netstat 
```


## 单行脚本

> 把 **命令** 看成是 **函数**，就特别好理解。

```bash
$ command && echo "TRUE" || echo "FALSE"
```
