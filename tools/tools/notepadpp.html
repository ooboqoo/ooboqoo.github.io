正则表达式替换字符串详解

EditPlus的查找，替换，文件中查找支持以下的正则表达式：

\t制表符
\r回车换行符
\n新行
\<匹配词头
\>匹配词尾
.匹配任意字符
|匹配表达式左边或右边的字符如 "(ab|bc)g" 匹配 "abg" 或 "bcg"
[...]匹配列表之中的任何单个字符如 "[A-Za-z]" 匹配所有大小写字母；"[0-9]" 匹配任意数字
[^...]匹配列表之外的任何单个字符如 "[^ab]" 匹配 a, b 以外的字符；"[^0-9]" 匹配非数字字符
*其左边的字符被匹配任意次(0次或多次)如 "be*" 匹配 b, be, bee 等
+其左边的字符被匹配至少一次(1次或多次)如 "be+" 匹配 be 或 bee 但不匹配 b
?其左边的字符被匹配0次或1次如 "be?" 匹配 b 或者 be 但是不匹配 bee
^行开始符，其右边的表达式被匹配在一行的开始如 "^A" 仅仅匹配以 A 开头的行
$行结尾符，其左边的表达式被匹配在一行的结尾如 "e$" 仅仅匹配以 e 结尾的行
( )影响表达式匹配的顺序，并且用作表达式的分组标记
\n(1-9)n可以是1到9，代表正在替换的第1到第9个匹配项
\转义字符如果你要使用 \ 本身, 则应该使用 "\\"；"\[" 表示[字符本身

应用举例:

1, 把 divv1/divv,divv2/divv,divv3/divv 替换成 1 2 3

查找 divv([0-3])/divv
替换 \1

说明: 替换中的\n(n是数字)，代表前面查找时候，正则的第几个匹配项（用括号表示）

说明：正则表达式不是notepad++专用的，以上列的是比较常用的，详细教程请点击 微软中文教程

在用插件介绍

Emmet

Emmet是一个文本编辑器的插件，它可以帮助您快速编写HTML和CSS代码。

Preview HTML

开启后会实时显示当前代码的最终显示效果，提供所见即所得的编辑体验

Light Explorer

轻量级的资源管理器。

NppExport

Notepad++的语法高亮拷贝到其他地方时会失效，该插件可以帮助导出已着色代码为其他格式的文件。

使用技巧及注意事项

1，自动完成和字符编码

小巧的notepad++也是有自动完成功能的，在Settings->Preference->Auto-Completion 里。
在web编程时，字符编码是很需要注意的一点，一般现在流行的都是utf-8编码，如果不做改变的话，默认为ANSI编码

2, Column Mode Editing

Using Alt + Mouse dragging or Alt + Shift + Arrow keys to switch to column mode

https://notepad-plus-plus.org/features/column-mode-editing.html

3, Moving And Selecting Around

具体的已经提炼并放到快捷键列表了

http://docs.notepad-plus-plus.org/index.php/Moving_And_Selecting_Around

4, 让notepad++支持less

notepad++直接打开.less文件的话，没有高亮提示，当然也没有zen coding了。
设置下吧：Setting >> Style Configurator >> CSS >> User ext. 输入框中填入less。
