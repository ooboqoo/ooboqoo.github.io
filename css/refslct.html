CSS Selectors

In CSS, selectors are patterns used to select the element(s) you want to style.

The "CSS" column indicates in which CSS version the property is defined (CSS1, CSS2, or CSS3).

CSS 标准参考：https://www.w3.org/TR/css-2015/

注意：在 W3C 标准 中，只有组合选择符符号两边有空格，其他地方都是不带空格的。CSS 中除了 > + ~ 和 , 的意义不受空格影响，其他地方有空格和没空格意义是完全不一样的（空格会被认为是“后代”选择符）。

各选择符优先级：id > class > attr > tag; 层叠优先级: 内联样式 > 内部样式表 > 外部样式表 > 浏览器缺省

后代选择符 descendant selector (space)
子选择符 child selector (>)

:target 示例

// CSS 代码
  #more_info:target {background:#eee;}
// HTML 代码
  <a href="#more_info">More Information</a>
  <h2 id="more_info">This is the information you are looking for.</h2>

会在用户单击链接转向 ID为 more_info 的元素时，为该元素添加浅灰色背景。
