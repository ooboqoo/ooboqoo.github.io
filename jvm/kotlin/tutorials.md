## Getting Started

### HelloKotlin

1. 创建项目 File -> new -> Project -> Kotlin -> Kotlin(JVM) -> HelloKotlin
2. 新建文件 src -> new -> Kotlin File/Class -> HelloKotlin + File
3. 录入代码 录入 main -> tab自动完成 -> println("Hello Kotlin!")
4. 运行 点击代码左侧图标 -> Run "HelloKotlinKt"
