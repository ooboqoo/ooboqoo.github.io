HTTP 服务器

apache lighttpd nginx

建议方案：
Apache 后台服务器（主要处理php及一些功能请求 如：中文url）
Nginx 前端服务器（利用它占用系统资源少得优势来处理静态页面大量请求）
Lighttpd 图片服务器
总体来说，随着nginx功能得完善将使他成为今后web server得主流。
