## 为什么要做笔记
前端开发世界空前繁荣，太多东西需要学习，凭借个人的记忆已经很难记住，所以学习毛泽东思想，打下一个地方就建立一个根据地，
我每学习一项就做好笔记，只有这样才能做到尽量把学习的内容记住，或者复习一下很快想起来。
另外，做笔记也是一个思考的过程，通过做笔记，可以促进对内容的分析和理解。总之，我认为笔记是个好东西，鸟哥的linux私房菜也是这么来的吧。
